# 🎵 Music Visualizer

 Real-time cyberpunk audio visualizer with ASCII art, matrix rain effects, and reactive waveforms.

## Features

- Real-time audio visualization
- Matrix-inspired ASCII rain
- Custom reactive ASCII flower art 
- Dynamic waveform effects
- Built-in music player controls
- Runs directly in the browser
- No installation required

  
## Built with 

- HTML5
- CSS3
- JavaScript
- Web Audio API

## How to Use

1. download the html file and open it in your browser.
2. Upload an audio file.
3. Watch the visualizer react to the music in real time.
